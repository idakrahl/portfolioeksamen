GRUNDLÆGGENDE ANIMATION

I ’grundlæggende animation’, blev vi introduceret til Javascript. JavaScript bruges bl.a. til at skabe interaktive elementer. I dette tema brugte vi JavaScript sammen med css animationer, til at skabe et spil. Vi lærte om formgivning af grafiske elementer, og vi blev introduceret for rentegning af grafik i Adobe Illustrator. Vi lærte at arbejde med aktivitets- og state machine-diagrammer, som bruges til at planlægge det interaktive flow.
